# Horse Racing Game

## Domain Rules

- 20 horses total, each with unique name, color, and condition score (1–100)
- 6 rounds per race: 1200m, 1400m, 1600m, 1800m, 2000m, 2200m
- 10 random horses selected per round
- Races run sequentially, one round at a time
- Results displayed after each round completes
- Horses animate visually during each race
